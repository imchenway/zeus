# Support

Use GitHub Issues for reproducible bugs and feature requests.

Before opening an issue, include:

- Zeus version or commit;
- macOS and CPU architecture;
- install path: Homebrew, DMG, ZIP, or source build;
- commands run, such as `pnpm verify:release` or `pnpm test:real-scan`;
- clear external wait states, such as missing AI CLI login, Telegram token, Apple signing, or notarization.

Do not include secrets, Bot Tokens, API keys, certificate files, private source code, or full sensitive terminal logs.
