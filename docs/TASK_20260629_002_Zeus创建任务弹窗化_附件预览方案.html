交互闭环

1横向排列附件保存后进入胶片条；多附件只在条内滚动。
2点击放大图片缩略图打开大图预览；非图片保持文件信息。
3关闭返回Esc、关闭按钮、点击空白都可关闭，焦点返回缩略图。
4详情复用创建后任务详情抽屉也用同一横向预览逻辑。

实现约束

A可访问缩略图是 button，alt 用附件名，弹层有 dialog 语义。
E失败态图片无法加载时显示“无法预览，本机路径已保存”。
S安全边界只读取本机 file 路径，不上传，不新增依赖。
能力验收口径风险控制
横向附件条图片/文件按添加顺序横向排列，条内滚动，页面不横向溢出移动端缩略图变窄，仍保持可点区域
放大预览点击图片打开大图，显示文件名和本机路径dialog focus trap、Esc、空白关闭、返回焦点
失败降级图片加载失败仍显示文件证据和删除操作不阻断创建任务
详情复用任务详情抽屉展示同样横向预览只读模式隐藏删除按钮
