2026-07-03 追加返修：无动作时禁止 More 死悬浮

默认空任务、无筛选、默认列状态下，更多 不再打开任何浮层；只有存在可执行动作时才渲染动作菜单。

  • 有搜索 / 状态 / 标签筛选时：显示 清除筛选。
  • 列显隐、顺序或宽度偏离默认时：显示 恢复默认列。
  • 默认状态：按钮禁用，不渲染 task-table-view-more-panel，避免死悬浮遮挡表格和空态。
