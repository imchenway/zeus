# TASK_20260723_001 删除 TDD 与单元测试

## 目标

用户明确要求删除 Zeus 项目现有 TDD 与单元测试体系，并要求后续不再恢复或新增这两类内容。本任务按用户最终指令直接执行，不再保留现有
Vitest 测试套件。

## 删除范围

- 删除 `apps/desktop/test`、`packages/*/test` 与 `scripts/*.test.*` 下的现有自动化测试文件及测试夹具。
- 删除 Vitest 配置、依赖、workspace 测试脚本和真实扫描测试脚本。
- 从 CI、发布脚本、贡献指南、README、PR 模板和当前设计/发布文档中移除测试门禁与 TDD 要求。

## 保留边界

- 历史任务文档与历史实施计划保留原始事实，不批量篡改过去已发生的 TDD/测试记录。

- 不修改本任务无关的用户工作区变更，不执行 git commit 或远端操作。
